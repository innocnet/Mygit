	like  模糊查询； %表示不确定字符； _表示一个字符
			eg:   where name like  '%a%'
				  where name like  '_a%'
				  where name lile  '%\_%'  escape '\'  (这里查询含有_字符的名字，其中'\'表示转义字符)

	order by   : asc顺序； desc 逆序(从大到小)
